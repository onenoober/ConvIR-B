# Haze4K DPGA Tail-Control Text Package

Date: 2026-06-04

Status: GitHub-readable sync package for the completed DPGA tail-control
diagnostic route.

## Purpose

This package is a compact reading entry point for the DPGA in-network
depth/prior-guided adapter route. It records what was run, why locked Haze4K
test evaluation is still blocked, and where the full text evidence lives.

## Contents

| File | Use |
| --- | --- |
| `ROUTE_SYNC_SUMMARY.md` | Gate-based result summary for runtime diagnostics, v1.1, and v1.2. |
| `ARTIFACT_MANIFEST.md` | Pointers to the full text evidence under `experience_docx/`. |

## Boundary

Included in GitHub: source code, shell launchers, Markdown summaries, JSON
gates, CSV per-image tables, text logs, and status files.

Excluded from GitHub: checkpoints, model weights, image outputs, datasets,
NumPy arrays, and raw inference artifacts.
